## 1. 소개
FMRS은 실제 축구 경기 통계와 Football Manager 게임의 선수 능력치를 통합하여 제공하는 웹 서비스입니다.
사용자는 이름, 소속팀, 국적, FM 능력치 등의 조건으로 선수를 검색하고, 상세 조회를 통해 해당 선수의 실제 경기 스탯(출전, 득점, 도움 등)과 매핑된 FM 능력치(현능력, 잠재능력, 기술·멘탈·피지컬 스탯 등)를 함께 확인할 수 있습니다.  
또한 인증된 사용자는 특정 선수에 대해 댓글을 작성 및 조회할 수 있으며, 필요에 따라 댓글을 삭제할 수 있습니다.  

## 2. 기술 스택
- Java 21
- Spring Boot 3.4.1
- Spring Data JPA
- Querydsl
- Spring Security, JWT
- AWS EC2
- MySQL

## 3. 주요 기능
- **데이터 동기화**: 최신화된 축구 데이터 제공
- **선수 검색**: 선수 이름 또는 상세 조건으로 검색
- **선수 상세 조회**: 특정 선수의 실제 경기 스탯과 FM 능력치 제공
- **유저 댓글**: 선수별 댓글 작성, 조회, 삭제
- **FM 데이터 제공**: 실제 선수와 FM 데이터를 매핑하여 제공
